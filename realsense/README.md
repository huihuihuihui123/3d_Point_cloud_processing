2.按下“空格”、“回车”、“q”键进行 深度图到点云的转换

3.在点云显示界面按下“空格键”，点云将保存在当前目录下

4.关闭点云显示的窗口后，回到深度图与RGB图显示主循环，再按ESC结束调试
